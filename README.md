# API-Manga
An API for Mangas
Live at http://webarranco.fr:1208

## Installation
	clone/pull the repository
	cd apiManga
	node synchro

## Launch
	supervisor -- app.js
	OR
	supervisor -- app.js --nointernet (if you don't have internet, mangas.json will be used)
